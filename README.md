# PICKle🍕

> Vue와 Django를 활용한 영화 추천 웹 서비스 개발 공간입니다.



## 🌈 About

- 개발 언어
  - Vue.js | Node 14.16.1
  - Python 3.8.7 | Django 3.2.3



## 🌟 Overview

- Key Concept
  - **PICK**
    - **PICK simpLE** 👉 간편하게 PICK하고
    - **PICK valuabLE** 👉 소중한 것을 PICK하고
    - **PICK articLE** 👉 다른 사람의 PICK이 담긴 글도 볼 수 있다!!
